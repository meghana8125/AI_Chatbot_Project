# 🤖 AI Chatbot using Google Gemini API

## 📌 Project Overview
This is an AI-powered chatbot developed using Python, Flask, HTML, CSS, JavaScript, and the Google Gemini API. The chatbot provides intelligent responses to user queries and supports image uploads for AI-based analysis.

## ✨ Features
- 💬 AI-powered chatbot
- 🖼️ Image upload support
- ⚡ Real-time responses using Google Gemini API
- 🎨 Responsive and user-friendly interface
- 🌙 Clean and modern UI

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Gemini API

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/meghana8125/AI-Chatbot-Project.git
```

2. Navigate to the project folder

```bash
cd AI-Chatbot-Project
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Add your Google Gemini API key in a `.env` or `api.env` file.

5. Run the application

```bash
python app.py
```

6. Open your browser and visit:

```
http://127.0.0.1:5000
```
